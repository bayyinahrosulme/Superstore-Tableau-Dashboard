# Superstore-Tableau-Dashboard
Interactive Tableau dashboard analyzing 10,000+ retail transactions.

### Tools Used
- Excel — Data cleaning and Pivot Tables
- Tableau Public — Interactive dashboard

### Dashboard Link
https://public.tableau.com/views/SuperstoresalesDashboard_17725796159850/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

### Key Findings
- Identified 3 underperforming sub-categories with profit margins below 5%
- Technology is the highest revenue category
- Sales spike significantly in November/December
